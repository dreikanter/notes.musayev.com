I'm updating gems regularly for \d\d years now, but it is still hard to remember related commands. Should I use gem or bundle command to upgrade bundle? Is it "update" or "upgrade"? Do I need to bump "system" gems, or usual bundle install will do in this case? And again, is it "bundle" or "bundler"? Bwagh!

Listen close!

Updating Bundler:

bundle update --bundler

Note: Gemfile.lock may lock Bundler version with BUNDLED WITH value. Also it may cause to alternative bundler version installation during bundle install.

Check current version with bundle --version.

Update system gems:

gem update --system
